# SPARK MVP Development Guide

## Overview
SPARK는 운동 기반 커뮤니티 플랫폼이다.

### 주요 기능
- 홈
- 운동 시작
- 번개 모임
- 챌린지
- 마이페이지

## Tech Stack
- React
- Vite
- TypeScript
- React Router
- Zustand
- TailwindCSS
- Supabase

## MVP Scope
### Auth
- Signup
- Login
- Logout

### Workout
- 운동 생성
- 운동 기록
- 운동 완료

### Meetup
- 모임 생성
- 모임 참여

### Challenge
- 챌린지 참여
- 진행률 관리

### Profile
- 프로필 수정
- 활동 내역 조회
